# 💫 About Me:

I am a developer focused on building practical software and automation tools using **Python**, **Laravel**, and modern **API-based architectures**. I specialize in designing backend systems, creating efficient APIs (Flask, FastAPI, Django, Laravel), and developing tools that solve real technical problems.

Much of my work involves developing **offensive security and red-team tools** for research and learning purposes. Notable projects include:
- **SpecterPanel**: A web-based Command & Control (C2) panel built with Flask, featuring encrypted API communication, botnet management, and code injection capabilities.
- **PhantomGate**: A persistent botnet agent that communicates with C2 servers, executes commands, performs UDP floods, and runs injected Python payloads.
- **GhostTrigger**: A multi‑purpose network utility (reverse shell, port scanner, FTP/HTTP servers, UDP flood) with a focus on security and educational use.
- **editer.py**: A minimalist command‑line code editor for quick Python scripting.

I also build **cross-platform applications for Android and desktop using Flet**, allowing me to turn backend logic and APIs into usable graphical tools. My projects usually combine **backend development, security concepts, and user‑friendly interfaces**.

I am always learning new technologies, improving my development workflow, and working on projects that expand my knowledge in **software engineering, security research, and system development**. 🚀
